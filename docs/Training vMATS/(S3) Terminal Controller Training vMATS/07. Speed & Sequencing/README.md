
# Speed & Sequencing

Speed control and sequencing are the primary tools used by approach controllers to establish and maintain an efficient arrival flow.

This section develops advanced techniques used to:
- Build and maintain spacing
- Manage traffic complexity
- Handle mixed aircraft performance
- Deliver stable approaches
