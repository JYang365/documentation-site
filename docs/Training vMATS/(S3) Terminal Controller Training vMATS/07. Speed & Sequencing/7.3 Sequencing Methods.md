# 7.3 Sequencing Methods

## Distance-Based Sequencing

- Maintain 3–5 NM spacing
- Most common method

---

## Time-Based Sequencing

- Used in high-density environments
- Requires consistent groundspeed

---

## Vector-Based Sequencing

- Use headings to build spacing
- Extend or shorten downwind

---

## Key Principle

Use the simplest method available.
