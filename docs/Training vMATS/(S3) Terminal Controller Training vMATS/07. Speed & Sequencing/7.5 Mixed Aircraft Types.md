# 7.5 Mixed Aircraft Types

Different aircraft types require different handling.

---

## Speed Differences

- Heavy jets: higher approach speeds
- Turboprops: slower and more variable

---

## Wake Turbulence

Controllers must apply appropriate spacing:
- Heavy → Medium
- Medium → Light

---

## Techniques

- Sequence similar aircraft together where possible
- Use speed control early
- Avoid compressing behind slower aircraft
