# 7.4 Arrival Spacing Models

## Basic Model

- Establish aircraft on downwind
- Build spacing on base leg
- Deliver stable final

---

## Continuous Flow Model

- Minimal vectoring
- Speed control dominant

---

## Extended Downwind Model

- Used when traffic demand exceeds capacity
- Allows spacing to be increased safely

---

## Controller Considerations

- Runway capacity
- Traffic volume
- Aircraft performance
