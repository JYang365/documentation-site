# Instrument Approaches

Instrument approaches are used to guide aircraft from the terminal area to landing using published procedures or radar vectoring.

This section covers:
- Types of approaches
- Controller responsibilities
- Approach clearances
- Missed approach handling

Controllers must ensure all approaches are:
- Stabilised
- Properly spaced
- Conducted safely
