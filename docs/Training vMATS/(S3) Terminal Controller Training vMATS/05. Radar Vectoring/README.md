# Radar Vectoring

Radar vectoring is the primary method used by approach controllers to position aircraft for sequencing and approach.

This section covers:
- Core vectoring techniques
- Intercept geometry
- Sequencing methods
- Practical application in a radar environment

Vectoring is a predictive skill and forms the foundation of safe and efficient approach control.
