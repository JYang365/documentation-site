# 5.4 Downwind & Base Leg

## Downwind Leg

The downwind leg is used to:
- Build spacing between aircraft
- Prepare for base turn

Controllers shall:
- Maintain consistent track spacing
- Adjust length based on traffic

---

## Base Leg

The base turn positions aircraft for final intercept.

Controllers must:
- Turn aircraft at the correct distance
- Ensure adequate spacing exists
- Consider aircraft speed

---

## Key Principle

Late base turns result in unstable approaches and increased controller workload.
