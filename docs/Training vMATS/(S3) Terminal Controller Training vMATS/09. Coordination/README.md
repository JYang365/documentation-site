# Coordination

Coordination between air traffic control units is essential to ensure safe, orderly, and efficient traffic flow.

This section defines standard coordination procedures between:
- Approach (APP)
- Tower (TWR)
- Area Control (ACC)

---

## Objectives

- Maintain separation across sector boundaries
- Ensure smooth transfer of control
- Minimise controller workload
- Maintain predictable traffic flow

---

## Key Principle

> Good coordination eliminates the need for last-minute correction.
